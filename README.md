# 阜阳容其商贸有限公司 官网

国际化企业官网，融合唐代盛世美学与现代商务设计。

## 预览

直接在浏览器中打开 `index.html` 即可预览：

```bash
open index.html
```

或启动本地服务器：

```bash
python3 -m http.server 8080
# 访问 http://localhost:8080
```

## 功能特性

- 中英双语切换（右上角 中/EN 按钮）
- 响应式设计，适配手机、平板、桌面
- 唐代风格视觉：金色、朱红、墨色、祥云纹样
- 板块：首页、关于我们、盛唐气韵、核心业务、大国风范、联系我们
- 联系表单（通过邮件客户端发送）
